## 🌟 功能特性

### 核心功能
- [x] **安全通信**
  - 所有传输数据使用 Base64 编码
  - 支持消息完整性验证
- [x] **文件传输**
  - 支持多文件并发上传
  - 断点续传功能（开发中）
- [x] **自动更新**
  - 版本检测机制
  - 静默下载更新包

### 技术指标
| 指标         | 参数               |
|--------------|--------------------|
| 协议         | TCP/IP             |
| 默认端口     | 54321              |
| 最大连接数   | 50                 |
| 数据编码     | Base64             |
| 数据库       | SQLite 3           |
